Aliyun API utils

## Installation

The package can be installed by adding `:aliyun_util` to your list of
dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:aliyun_util, "~> 0.3.5"}
  ]
end
```

## Usage

Available utility helper functions:

* `Aliyun.Util.Encoder`
* `Aliyun.Util.Sign`
* `Aliyun.Util.Time`
